# Repo Snapshots

## Papel

Preserva snapshots historicos de saidas ou arvores do repositorio quando houver valor de comparacao ou auditoria.

## Regra

- Nao e fonte de build.
- Nao e fonte de codigo vivo.
- Nao receber snapshots sem origem e motivo claros.
- Evidencias sensiveis de producao nao entram aqui.

## Conteudo classificado

- `frontend-build-prod-20260408-143932/`: snapshot de saida de build do frontend preservado para comparacao.
- `frontend-build-prod-20260408-150140/`: snapshot de saida de build do frontend preservado para comparacao.
- `frontend-build-prod-20260408-151038/`: snapshot de saida de build do frontend preservado para comparacao.
- `frontend-build-prod-20260408-aba-tabs-removed/`: snapshot de saida de build do frontend preservado para comparacao.
- `frontend-build-validate-20260408-aba-tabs-remove/`: snapshot de validacao de build do frontend preservado para comparacao.
- `frontend-prod-hotfix-20260408-144520/`: snapshot de saida/hotfix do frontend preservado para auditoria.
