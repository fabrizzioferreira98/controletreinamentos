## 3. Mudanca aplicada

Novos modulos:

| modulo | responsabilidade |
| --- | --- |
| `frontend/src/shell/render-shell.js` | shell autenticado, flash renderizado, menu/sidebar/topbar, interacoes do menu e logout. |
| `frontend/src/shell/navigation.js` | `NAV_GROUPS`, filtro por permissao, resolucao de item ativo e render dos links ja classificados. |
| `frontend/src/shell/login.js` | login SPA, refresh cacheado de sessao/CSRF, submit autenticado, retry de CSRF, feedback e destino pos-login. |
| `frontend/src/shell/redirects.js` | `FRONTEND_HASH_BY_BACKEND_PATH` e `resolveLoginDestination()` para compat backend/hash. |

`frontend/src/shell.js` permanece como facade temporaria para preservar imports existentes:

```js
export { renderLoginPage } from "./shell/login.js?v=20260404-1215";
export { renderShell } from "./shell/render-shell.js?v=20260404-1215";
```

O bloco nao removeu links, nao alterou classificacao B04 e nao mudou pertencimento de rota. O router oficial continua em `frontend/src/app/route-registry.js`.
