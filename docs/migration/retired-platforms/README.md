# Plataformas retiradas

Esta pasta guarda artefatos de plataformas descontinuadas. Eles nao fazem parte da operacao oficial self-hosted Windows.

Nao use estes arquivos como fonte de deploy, runtime ou build. Mudancas aqui devem se limitar a preservacao historica ou aviso de retirada.

Registros historicos `35.tv-panel-*.md` foram rebaixados para esta pasta durante a limpeza de reporting/PDF/calculos da remocao de Painel TV e Produtividade.
