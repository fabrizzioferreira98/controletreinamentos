## 5. Shared UI baseline

Primitives iniciais:

| primitive | uso permitido |
| --- | --- |
| `ui-stack`, `ui-stack-xs`, `ui-stack-sm` | composicao vertical neutra. |
| `ui-cluster` | composicao horizontal flexivel. |
| `ui-panel-offset`, `ui-panel-offset-sm`, `ui-block-end-sm` | spacing reutilizavel sem inline style. |
| `ui-heading-reset`, `ui-subtitle-compact` | normalizacao minima de titulo/subtitulo em blocos existentes. |
| `ui-surface` | superficie semantica basica para futura adocao. |
| `ui-feedback` | estado visual neutro por `data-kind`. |
| `ui-badge` | marcador compacto neutro. |

Regra: primitives nao conhecem dominio, nao chamam API, nao importam feature/shell/router e nao substituem componentizacao futura.
