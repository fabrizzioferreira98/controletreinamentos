## 3. Mudanca aplicada

Foi alterado `tests/contract/test_frontend_ux_functional_guards.py` para:

- remover a constante que apontava para `ROOT/frontend/dist`;
- gerar o artefato final por fixture usando `frontend/scripts/build_frontend.py --output-dir <tmp_path>`;
- validar o artefato final gerado sem depender de `frontend/dist` persistido;
- manter as leituras de `frontend/src` como fonte viva para as assercoes de comportamento.

Foi alterado `docs/architecture/FRONTEND_ARCHITECTURE.md` para registrar que checks automatizados que precisam inspecionar o artefato final devem preferir `--output-dir <diretorio-temporario-fora-do-checkout>`.
