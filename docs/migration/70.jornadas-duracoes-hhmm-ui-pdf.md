# 70 - Jornadas com duracoes em HH:MM na tela e no PDF

## Escopo

Correcao de apresentacao de duracoes da aba `Lancamentos de Jornada`.

Foco:

- hora reduzida;
- horas diurnas e noturnas exibidas na grade;
- adicionais pre/pos exibidos como duracao;
- resumo e PDF de Jornadas.

Fora de escopo:

- formulas de calculo;
- contratos;
- endpoints;
- persistencia;
- regras de feriado, sabado ou virada de dia.

## Problema identificado

A tela e o PDF exibiam duracoes em formato decimal, por exemplo:

```text
4,86 h
165 min
```

Esse formato e pouco operacional para conferencia de jornada, especialmente em casos de virada de dia, adicional pre/pos e hora reduzida.

## Mudanca aplicada

Foi adicionada uma camada de formatacao de duracao:

- horas decimais passam a ser exibidas como `HH:MM`;
- minutos passam a ser exibidos como `HH:MM`;
- valores internos continuam iguais para calculo, API e persistencia.

Exemplos:

```text
4,86 h -> 04:52
1,50 h -> 01:30
165 min -> 02:45
```

## Preservacoes funcionais

- Nenhum calculo foi alterado.
- Nenhum contrato foi alterado.
- Nenhum endpoint foi alterado.
- Nenhuma persistencia foi alterada.
- A alteracao e apenas de exibicao.
