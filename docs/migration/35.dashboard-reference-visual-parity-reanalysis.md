## Mudanca aplicada

Ajustes materiais para aproximar a tela real do mock de referencia (`WhatsApp Image 2026-04-26 at 12.52.36.jpeg`), sem abrir redesign lateral:

- faixa superior refinada:
  - copy do titulo/subtitulo alinhada ao mock;
  - label de acoes rapidas em caixa alta;
  - grid das acoes rapidas estabilizado em quatro CTAs no desktop.
- camada KPI ajustada para paridade:
  - percentual dos cards calculado sobre `base monitorada` (nao sobre soma parcial dos KPIs);
  - remocao do estado textual redundante (`com pendencia`/`sem pendencia`);
  - adicao de sparkline por card para leitura rapida;
  - CTA de card com seta no rodape.
- `Fila critica` calibrada:
  - cabecalho da ultima coluna alinhado para `Acao`;
  - densidade visual do bloco mantida como painel operacional central.
- `Base operacional` com card system mais fiel:
  - hierarquia `label -> numero -> apoio -> CTA` consolidada;
  - icone por card com superficie dedicada;
  - remocao de ruidos herdados de card anterior (barra lateral/efeitos).
