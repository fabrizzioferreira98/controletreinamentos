# Documentacao de Migracao

## Papel

Esta area guarda registros de transicao que explicam mudancas de modelo, plataforma ou arquitetura.

## Regra

- Nao define operacao oficial.
- Nao substitui `docs/architecture`, `docs/operations`, `docs/governance` ou `docs/product`.
- Material de plataforma retirada deve ficar em `docs/migration/retired-platforms`.
