# 75 - Jornadas: status da grade sem ruido tecnico de fallback de preview

## Contexto

A grade de `Lancamentos de Jornada` exibia na coluna `Status` o aviso tecnico:

`Resultado persistido de calculo nao encontrado; calculo preview aplicado.`

Esse texto e util para diagnostico interno, mas nao representa uma pendencia operacional para o usuario quando o calculo de preview foi aplicado corretamente.

## Mudanca

- Filtragem local no frontend para ocultar esse aviso tecnico da coluna `Status`.
- Preservacao de erros e avisos operacionais reais, que continuam colapsaveis em `Ver detalhes`.
- Preservacao das flags operacionais (`CAF`, `ALM`, `JAN`, `CEI`, `Descanso`) na coluna `Flags`.
- Ajuste visual local da coluna `Status` para reduzir truncamento de labels como `1 aviso` e `2 erros`.
