# 72 - PDF de Jornadas com contraste corrigido em cabecalhos

## Escopo

Hotfix de UX/UI no PDF de `Lancamentos de Jornada`.

Foco:

- contraste de texto em cabecalhos de tabelas;
- legibilidade de fundos escuros;
- maturidade visual do documento.

Fora de escopo:

- calculos;
- contratos;
- endpoints;
- persistencia;
- UI da tela.

## Problema identificado

Alguns cabecalhos do PDF usavam fundo escuro com `Paragraph` herdando estilo de texto escuro.

Em ReportLab, o `TEXTCOLOR` aplicado via `TableStyle` nao sobrescreve a cor interna de um `Paragraph`. Resultado: letras escuras sobre fundo escuro.

## Mudanca aplicada

Foi criado o estilo:

```text
JornadasTableHeaderLabel
```

Esse estilo usa texto branco e passou a ser aplicado nos cabecalhos escuros das tabelas do PDF de Jornadas:

- bases de adicionais;
- grade detalhada;
- subtotais por tripulante;
- fechamento por tipo de dia.

## Preservacoes funcionais

- Nenhum calculo foi alterado.
- Nenhum dado do PDF foi alterado.
- Nenhum endpoint foi alterado.
- Nenhuma persistencia foi alterada.
