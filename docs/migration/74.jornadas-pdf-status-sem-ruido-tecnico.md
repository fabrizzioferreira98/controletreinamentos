# 74 - PDF de Jornadas sem ruido tecnico no status

## Escopo

Hotfix de UX textual no PDF de `Lancamentos de Jornada`.

Foco:

- remover aviso tecnico de preview do relatorio final;
- rotular flags operacionais;
- nao exibir justificativa placeholder como `N/A`.

Fora de escopo:

- calculos;
- contratos;
- endpoints;
- persistencia;
- UI da tela.

## Problema identificado

O PDF podia exibir no status da linha:

```text
Aviso: Resultado persistido de calculo nao encontrado; calculo preview aplicado.
ALM, JAN
Just.: N/A
```

Esse texto mistura detalhe tecnico interno com informacao operacional e ainda imprime `N/A` como se fosse justificativa real.

## Mudanca aplicada

- O aviso tecnico `Resultado persistido de calculo nao encontrado; calculo preview aplicado.` deixou de aparecer no PDF.
- Flags passaram a aparecer como `Alertas: ALM, JAN`.
- Justificativas placeholder (`N/A`, `-`, vazio, `nao se aplica`) deixaram de ser renderizadas.
- Erros reais e avisos operacionais continuam aparecendo.

## Preservacoes funcionais

- Nenhum calculo foi alterado.
- Nenhum contrato foi alterado.
- Nenhum endpoint foi alterado.
- Nenhuma persistencia foi alterada.
